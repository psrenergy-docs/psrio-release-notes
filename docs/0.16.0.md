## 0.16.0 (2022-09-21)

- `Tab` class
- dashboard submenu support
- `dashboard:set_icon(url)` and `dashboard:set_logo(url)`
- `exp:spread_blocks(blocks)`

- data literal when using a `--model none`
- `exp:to_block(f, "hblock file")`
- `exp:spread_stages()`

- `Dashboard` concatenation operator (!)

### Submodules
