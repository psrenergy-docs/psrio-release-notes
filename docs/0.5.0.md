## 0.5.0 (2021-01-27)

### Added
- Lua
- dashboard support
- `--horizon <date>` to change to initial date
- `--model none` to load an empty study

### Syntax Changes
- several breaking changes due to Lua introduction

### Removed
- bison scanner and parser
- verbose 1f, 2f, and 3f (use 1, 2, or 3)
